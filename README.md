# bulls-and-cows
Игра "быки и коровы" написанная на с++. Код игры состоит из 3 файлов main.cpp br.cpp br.h. Запуск игры производиться через запуск .exe файла. В игре предусмотренно 3 режима.
1. Игра на 4 знака. (пользователь угадывает число компьютера)
2. Игра на 6 знаков. (пользователь угадывает число компьютера)
3. Компьютер угадывает число пользователя.


Ввод данных в игре осуществлен 3 способами:
1. С клавиатуры
2. С помошью рандома
3. Из файла
